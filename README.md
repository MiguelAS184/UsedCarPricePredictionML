# UsedCarPricePredictionML
Built regression models to predict used car prices using Python, scikit-learn, and feature engineering techniques.

# Machine Learning Algorithms Project

This project explores multiple machine learning techniques using scikit-learn.

## Models Used
- Random Forest
- K-Means Clustering
- PCA (Dimensionality Reduction)
- t-SNE

## Tools & Libraries
- Python
- pandas, NumPy
- scikit-learn
- matplotlib / seaborn

## Key Outcomes
- Compared supervised vs unsupervised models
- Evaluated feature importance
- Visualized high-dimensional data
